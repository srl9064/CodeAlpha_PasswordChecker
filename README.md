## Description
This project is a simple Python-based Password Strength Checker.
It analyzes a password entered by the user and classifies it as:
Very Weak, Weak, Medium, or Strong.

## Features
- Checks minimum password length
- Validates uppercase and lowercase letters
- Detects numbers and special characters
- Provides suggestions to improve password strength

## How to Run
```bash
python3 password_checker.py

## Sample Output
Enter your password and receive strength feedback along with improvement suggestions.
