This project is developed as **Task 2** of the **CodeAlpha Cybersecurity Internship**.  
It is a Python-based **Password Strength Checker** that evaluates the strength of a user-provided password and gives suggestions to improve it.

---

##  Project Overview

The Password Strength Checker analyzes a password based on common security rules such as:

- Password length  
- Use of uppercase letters  
- Use of lowercase letters  
- Use of numbers  
- Use of special characters  

Based on these checks, it classifies the password strength and provides feedback.

---

##  Features

- Checks password strength interactively  
- Classifies passwords as Very Weak / Weak / Medium / Strong  
- Provides improvement suggestions  
- Beginner-friendly and easy to use  
- Runs on Ubuntu / WSL
